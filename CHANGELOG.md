## [1.0.0] — 2026-03-05

### Добавлено
- ✅ Базовая структура Next.js 16 с App Router
- ✅ Интеграция shadcn/ui компонентов
- ✅ Tailwind CSS 4 для стилизации
- ✅ TypeScript конфигурация
- ✅ Prisma ORM для работы с БД
- ✅ NextAuth.js для аутентификации
- ✅ Zustand для управления состоянием
- ✅ TanStack Query для работы с данными
- ✅ Поддержка тёмной темы
- ✅ Адаптивный дизайн
- ✅ .gitignore с правилами для ROM файлов
- ✅ Docker и docker-compose конфигурация
- ✅ Документация (README, CONTRIBUTING, LICENSE)

### Изменено
- 📝 Обновлён README с информацией об авторе и школе Maestro7IT
- 📄 Добавлена строгая лицензия (UNLICENSED)
- 📦 Обновлён package.json с корректными метаданными

### Техническое
- 🔧 Настроена сборка через Bun
- 🔧 Скрипты для работы с Prisma (db:generate, db:push, db:migrate)
- 🔧 Production сборка с standalone режимом
